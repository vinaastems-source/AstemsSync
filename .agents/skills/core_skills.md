# 핵심 기술 역량 정의 (Core Skills)

AstemsSync 개발에 필요한 핵심 기술 및 구현 방법론 모음입니다.

## 1. Tauri IPC 통신 프로토콜
* 정의: FF(JS)와 FB(Rust) 간의 비동기 커맨드 호출 및 이벤트 감시 기술
* 활용: 로그인 요청 전달, 다운로드 진행률 실시간 갱신 이벤트 송신

## 2. 안전한 파일 교체 (Safe Overwrite)
* 정의: 실행 중인 프로그램 파일의 안정적인 교체 및 백업 기술
* 활용: 신규 버전 적용 시 파일 잠금 해제 확인 및 순차적 교체

## 3. 배포 스케줄링 알고리즘 (Rollout Algorithm)
* 정의: 매장 리스트에서 특정 비율이나 그룹을 추출하여 단계별 배포를 수행하는 로직
* 활용: 100개 매장 중 5개 선배포 등 카나리(Canary) 배포 구현

## 4. 네트워크 복원력 (Network Resilience)
* 정의: 불안정한 네트워크 환경에서 끊긴 시점부터 다운로드를 재개하거나 지능적으로 재시도하는 기술
* 활용: 파일 다운로드 중 단절 시 처리 로직
